## What is this place?

This repository is more than just a collection of files — it's a journal. A record of late nights, small victories, frustrating bugs, and the quiet satisfaction of finally making something *work*.

Every project here represents a moment where I chose to sit down, push through the confusion, and build something from nothing. That takes courage. And this repo is proof that I showed up.

## Why I built this

Learning Python wasn't just about learning a language. It was about proving something to myself — that I could think logically, solve problems, and grow beyond where I started.

I built this repository to:

- Strengthen my **fundamentals**, one concept at a time
- Create a space where **progress is visible** — not just to others, but to myself
- Build a habit of **writing code every day**, even when it's messy
- Look back someday and see **how far I've come**
